1. Al momento la gestione dei file avverrà LATO COMOL nel code-Behind delle pagine
2. Il path sarà del tipo: /File/#SkinId/
3. Css:
	/File/Skin/#SkinId/default.css
	/File/Skin/#SkinId/admin.css
	/File/Skin/#SkinId/IE.css
4. Loghi
	/File/Skin/#SkinId/#LogoID_OriginalName.ext
5. Altri file
	/File/Skin/#SkinId/Images/*img.ext
